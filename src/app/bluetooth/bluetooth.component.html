Zugriff auf Bluetooth

Leider ist auf diesem Gerät der Zugriff auf die Web Bluetooth API nicht möglich.

Damit der Bluetooth-Zugriff funktioniert, musst du ein paar Dinge beachten. Stelle sicher, dass Bluetooth und der Standort auf deinem Gerät eingeschaltet sind. Verbinde dich zunächst außerhalb der App mit einem Bluetooth-Low-Energy-Gerät, ansonsten ist dieses für die Web Bluetooth API häufig nicht auffindbar. Klicke anschließend den Button "Mit Gerät verbinden". Du solltest das Gerät nun in einer Liste finden und kannst dich damit verbinden. Falls das Gerät einen Battery-Service anbietet, wird dir der aktuelle Ladestand des Geräts angezeigt.

Der Ladestand des Geräts ist: {{battery}}%

Der Ladestand des Geräts ist: 100%
